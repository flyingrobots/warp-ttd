# Receipt Flame Graph

Receipts have flame-graph-like structure: each tick is a row, each
op within the tick is a segment, color-coded by outcome (admitted,
rejected, superseded, redundant).

A terminal-rendered flame graph would instantly show hotspots of
conflict, which writers are contending, and where the interesting
causal decisions happened. Pairs naturally with the braille
adjacency idea for graph density.
