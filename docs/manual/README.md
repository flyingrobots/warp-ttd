# WARP TTD Manual Index

This directory is the maintained manual for WARP TTD. It is not a replacement
for `docs/design/`; it is the compiled, durable reading path over design
decisions that should survive beyond an individual cycle.

## How To Use This Manual

- Start here when you need the current operating model rather than a historical
  design packet.
- Follow each chapter's source design link when you need playback questions,
  cycle scope, or implementation evidence.
- Add or update a chapter for every new design cycle that changes durable
  doctrine, architecture, protocol boundaries, operator workflows, or agent
  contracts.
